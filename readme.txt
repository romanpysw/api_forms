api_json_rpc.py

    Исходный код API
    Предназначен для реализации функционала создания форм и получения ответов на них

    Разработан с использованием FastAPI, databases

    Тестирования проходили на веб-сервере uvicorn

    Взаимодействует с СУБД sqlite


models.py

    Описание моделей-сущностей базы данных (not ORM)

    Хранит описнаие моделей сущностей с использованием классов и наследования от BaseModel pydantic
        pydantic -- модуль Python, идет вместе с FastAPI


sqlite/form.db

    Файл базы данных sqlite


Детальное описание взаимодействия API и базы данных демонстрируется в "Диаграмма взаимодействий.png"
